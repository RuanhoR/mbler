# MBLER

### 소개

Rooldown 기반의 Minecraft Addon 개발 도구입니다. 다음과 같은 기능이 있습니다:

- MCX DSL
- Manifest Generator
- Bundle
- Component build

### 설치

다음 도구가 설치되어 있는지 확인하세요:

- Node.js와 npm
- Git
- npm
  `npm install -g mbler`

설치 후 직접 사용할 수 있습니다:

```bash
# 버전 확인
mbler -v
```

디렉토리 설정 파일 초기화

```bash
mbler init
```

자세한 내용은 [문서](https://mbler-docs.ruanhor.dpdns.org/)를 참조하세요.
