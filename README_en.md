# MBLER

### Introduction

A development tool for Minecraft Addon based on Rooldown, featuring:

- MCX DSL
- Manifest Generator
- Bundle
- Component build

### Installation

Ensure you have the following tools installed:

- Node.js and npm
- Git
- npm
  `npm install -g mbler`

After installation, you can use it directly:

```bash
# Get version
mbler -v
```

Initialize directory configuration file

```bash
mbler init
```

For details, see: [Documentation](https://mbler-docs.ruanhor.dpdns.org/)
