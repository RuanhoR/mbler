# MBLER

### 紹介

RooldownベースのMinecraft Addon開発ツールです。以下の機能があります:

- MCX DSL
- Manifest Generator
- Bundle
- Component build

### インストール

以下のツールがインストールされていることを確認してください:

- Node.jsとnpm
- Git
- npm
  `npm install -g mbler`

インストール後、直接使用できます:

```bash
# バージョンを確認
mbler -v
```

ディレクトリ設定ファイルを初期化

```bash
mbler init
```

詳細は [ドキュメント](https://mbler-docs.ruanhor.dpdns.org/) をご覧ください。
