# MBLER

### 介绍

一个Minecraft Addon的开发工具，功能包括
 - MCX DSL
 - Manfiest Generator
 - Bundle
 - Component build

### 安装

确保你已经安装以下工具：

- Node.js 与 npm
- Git
- npm  
`npm install -g mbler`
  安装成功后，你可以直接使用：

```bash
# 获取版本
mbler -v
```

初始化目录配置文件

```bash
mbler -c <项目路径>
mbler init
```

** 详细请看 : [文档](https://mbler-docs.ruanhor.dpdns.org/) **
