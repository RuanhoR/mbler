# v0.0.4 (2025-11-20)

Initial release of mbler — a CLI tool for Minecraft Bedrock addon development.

### Features

- **Project initialization**: Scaffold new addon projects with `mbler init`
- **Build system**: Bundle addon scripts and assets with `mbler build`
- **Watch mode**: Automatic rebuild on file changes with `mbler watch`
- **Manifest generation**: Auto-generate `manifest.json` for behavior/resource packs
- **SAPI version management**: Fetch and cache Minecraft Script API module definitions
- **Built-in modules**: Support for `gameLib` and `sapi-game` runtime modules
- **Module loading**: Resolve and install npm dependencies for addon projects
- **JSON minification**: Minify JSON files during production builds
- **Zip packaging**: Package addon into `.mcaddon` / `.mcpack` format
