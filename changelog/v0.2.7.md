# v0.2.7 (2026-05-25)

### Features

- **Rolldown support**: Replaced Rollup with Rolldown for faster builds; removed basic Rollup plugin since Rolldown covers it natively
- **`use create-mbler`**: Init command now uses `create-mbler` package for scaffolding; added `packageManager` option in generated `package.json`
- **Build cache enhancement**: Custom cache paths, improved mode resolution, `rollupExternal` and `cachePath` in `MblerBuildConfig`
- **SAPI module refactor**: Refactored Sapi module to use Proxy for lazy initialization and state management
- **Release generation**: New `Release` function encapsulating zip creation; cleaner release workflow
- **Git hooks**: Added `simple-git-hooks` configuration
- **Multi-language README**: Added English, Japanese, Korean README translations
- **MCX template package**: Template for MCX language projects during init
