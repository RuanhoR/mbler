# v0.0.6 (2025-12-18)

### Features

- **Multi-language support**: Internationalization (i18n) framework with English and Chinese language packs
- **`mbler create` command**: Create projects from templates
- **`mbler install` command**: Auto-detect `mbler.config.json` and install dependencies
- **Command prediction**: Typo-tolerant command matching using character-position comparison algorithm (e.g. `mbler buid` suggests `mbler build`)
- **UUID generation**: Resource pack UUIDs now use salted hashing for uniqueness
- **Project restructuring**: Separated behavior pack and resource pack directories; layered manifest structure
- **Line statistics**: New `lines` command for project statistics

### Fixes

- Fixed zip packaging root directory incorrect after build
- Fixed newly created English language pack not switching properly
- Fixed `create` command file system access in VM environment
- Fixed command recommendation method naming error
